About the Environment

This is a stripped down version of JohnPapa's gulp-patterns. It is intended for developers who want to create an Angular environment from the ground up. It contains a variety of basic tools to allow the developer to hit the ground running.

Included Tools

  • Preconfigured Gulpfile
  • Gulp Configuration File
  • DefinitelyTyped TypeScript definitions
  • Bower Configuration Definitions
  • JSHint Code Quality Tool
  • JavaScript Code Style Linter
